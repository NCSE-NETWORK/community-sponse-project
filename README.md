# NCSE 社群贊助項目網站

## 專案簡介

本網站用於展示 NCSE 社群的贊助項目、合作社群及聯絡資訊，協助推廣社群資源與服務。

## 主要功能

- 贊助項目介紹（VPS、網頁主機、技術支援、Public IP tunnel）
- 合作社群展示
- 聯絡資訊與申請流程
- 響應式設計，支援各種裝置

## 技術棧

- HTML5
- Tailwind CSS
- AOS (Animate On Scroll)
- EmailJS（如有聯絡表單）

## 如何啟動

1. 下載或 clone 本專案
2. 直接以瀏覽器開啟 `index.html` 即可瀏覽
